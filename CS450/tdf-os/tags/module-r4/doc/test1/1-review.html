1. Why is an operating system such an important part of most computing systems? 

The operating system allows everything to run smoothly in a system by completely

managing the resources of the system for a user.


   2. State the principal purpose of an operating system in a single sentence. 


The operating system serves to manage the resources of a system.


   3. Explain briefly the two principal objectives of resource management by operating 

systems. 


Convenient Use: Most users of a computer system aren't programmers and want

all of the background work to stay in the background, so users don't have to worry

about complex details involved with the system.


Controlled Sharing: To prevent programs from interfering with each other.
